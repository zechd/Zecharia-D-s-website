# Personal Website Starter

A deliberately blank starting point for building your own website by prompting an
AI. Plain HTML and CSS — no build step, nothing to install.

- `index.html` — your page (starts nearly empty on purpose).
- `style.css` — a clean, minimal base you'll shape with prompts.

**Students:** follow the class handout. The short version:

1. **Use this template** on GitHub to make your own copy.
2. Turn on **GitHub Pages** (Settings → Pages) to get your live web address.
3. Open your copy in **Replit**, click **Run** to preview, and start prompting the
   AI to build your site.
4. **Push** your changes — GitHub Pages updates your live site automatically.

You can't break anything. Experiment, run it, see what happens, adjust.
